timelinefx:timelinefx.tlEffect.GetEmitter
Method GetEmitter:tlEmitter( name:String )
Retrieve an emitter from the directory of the effect Use this To get an emitter from the effect directory by passing the name of the emitter you want. All effects And emitters are stored using a map with a directory like path structure. So retrieving an emitter called blast wave inside an effect called explosion would be done like so: 
local blastwave:=MyEffectsLibrary.GetEmitter("explosion/blast wave")
Note that you should always use forward slashes.
Parameters
name the name of the emitter
