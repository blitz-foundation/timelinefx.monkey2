timelinefx:timelinefx.CreateLine
Function CreateLine:tlLine( x1:Float, y1:Float, x2:Float, y2:Float, layer:Int=0, data:@object=Null )
Create a tlCircle Create a new tlLine at the coordinates given, x1 and y1 being the start of the line and x2 and y2 being the end. The will placed exactly according to the coordinates you give, but it's worth bearing in mind that the handle of the line will be at the center point along the line. Therefore the world coordinates will be set to half way point along the line. Use layer to specify a particular layer to place the box on so that you can more easily organise your collisions. You use 0, 1..and so on up to 31. to place the boundary on all layers.

@example Local box:=CreateLine(0, 0, 50, 100, 0) @end

Parameters
x1 First x point of the line
y1 First y point of the line
x2 Second x point of the line
y2 Second y point of the line
layer The layer that the box is on, used for quadtrees
data any Object associated with the box
