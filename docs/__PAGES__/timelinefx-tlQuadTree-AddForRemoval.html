timelinefx:timelinefx.tlQuadTree.AddForRemoval
Method AddForRemoval:Void( r:tlBox )
Add the tlBox to the remove list This will add a tlBox to the queue for removal next time you call CleanUp. It's important to use this if you're removing items from the quadtree during a query because otherwise you will run into a "concurrent list modification error". At the end of your update loop you can call CleanUp to remove those items from the quadtree in a batch. Note that if you are using a tlGameObject then tlGameObject.Destroy will do this automatically, but you'll still need to call CleanUp
