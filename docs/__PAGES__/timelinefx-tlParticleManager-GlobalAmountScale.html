timelinefx:timelinefx.tlParticleManager.GlobalAmountScale
Property GlobalAmountScale:Float
The Global amount scale is used to control the overal number of particles that will be spawned.. The default value is 1, meaning that the particle manage will spawn the same number of particles as designed in the editor. To spawn half the amount of particles, set to 0.5. This feature is useful for giving users the option to tone down the number of particles if their device is running slowly. 
MyParticleManager.GlobalAmountScale(0.5)
