timelinefx:timelinefx.tlBox.PreventOverlap
Method PreventOverlap:Void( result:tlCollisionResult, push:Int=false )
Prevent the boundary from overlapping another based on the result of a collision. When you check for a collision, the results of that collision are stored with a tlCollisionResult. This can be passed to this method to prevent 2 boundaries from overlapping. If push is set to true, then the source boundary will push the target boundary along it's velocity vector.
Parameters
result The tlCollisionResult with the collison data to use to move the objects apart
push Set to true if you want the box to push away the colliding object
