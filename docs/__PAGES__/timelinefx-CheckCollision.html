timelinefx:timelinefx.CheckCollision
Function CheckCollision:tlCollisionResult( source:tlBox, target:tlBox )
Check for a collision between 2 Boundaries You can use this function to check for collisions between any type of boundary: tlBox, tlCircle, tlLine and tlPolygon. The tlCollisionResult can then be used to determine what you want to do if a collision happened (or will happen). See PreventOverlap to make boundaries block or push each other.
Parameters
source tlBox object
target tlBox object
