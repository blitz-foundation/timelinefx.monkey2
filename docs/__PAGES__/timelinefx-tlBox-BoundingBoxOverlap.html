timelinefx:timelinefx.tlBox.BoundingBoxOverlap
Method BoundingBoxOverlap:Int( rect:tlBox, velocitycheck:Int=false )
Compare this tlBox with another to see if they overlap Use this to find out if this tlBox overlaps the tlBox you pass to it. This is a very simple overlap to see if the bounding box overlaps only Set VelocityCheck to true if you want to see if they will overlap next frame based on their velocities.
Parameters
rect tlBox to check with
